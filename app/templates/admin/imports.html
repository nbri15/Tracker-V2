Combined CSV import Recommended
Download combined template

Upload one row per pupil. Pupil details are created or updated, blank subject cells are ignored, and manual teacher-entered results are protected by default.

Combined template columns
  • Pupil fields: first_name, last_name, gender, pupil_premium, laps, service_child, class_name, academic_year.
  • Optional subjects: maths, reading, and SPaG paper columns for autumn, spring, and summer.
  • Optional writing: writing_{{ '{term}' }}_band and writing_{{ '{term}' }}_notes.
Import rules
  • Pupils are matched by first_name + last_name + class_name.
  • If both subject paper cells are blank, that subject/term is ignored safely.
  • If one score is provided, the import keeps any existing CSV score in the other paper where possible.
  • Existing manual/protected results are skipped and listed in the summary.
